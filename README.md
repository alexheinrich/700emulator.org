# 700 emulator

A site to showcase the Buchla 700 synthesizer.

## Running the project
```
npm install
```

```
npx eleventy
```

Or build and host locally for local development
```
npx eleventy --serve
```

Or build automatically when a template changes:
```
npx eleventy --watch
```

Or in debug mode:
```
DEBUG=* npx eleventy
```
